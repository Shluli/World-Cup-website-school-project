# Uri'sWorldCups

Uri'sWorldCups is a web application focused on the history of the FIFA World Cup.  
The project presents information about different World Cup tournaments, including descriptions, facts, and visual elements, while also supporting user accounts and basic admin functionality.

This project was originally built as a final course project and is being uploaded for documentation and portfolio purposes.

## Features
- Homepage with navigation to individual World Cup tournaments
- Dedicated pages for each World Cup edition with:
  - Tournament description
  - Interesting facts
  - Images of knockout and group stages
- User system:
  - Registration and login
  - Logout
  - Update personal details
  - Delete account
- Admin area for managing users

## Tech Stack
- ASP.NET Web Forms
- SQL
- C#
- HTML
- CSS
- JavaScript
- Developed using Visual Studio 2022

## Database
The project uses a simple user database with the following fields:
- username
- password
- fullname
- gmail

## Project Structure
- `WorldCups.aspx` – Main page
- `WorldCupYYYY.aspx` – Individual World Cup tournament pages
- `AdminPage.aspx` / `admin.aspx` – Admin interface
- ASP pages for database operations (update, delete)
